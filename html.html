¿Què es HTML?



HTML (HyperText Markup Language) es el lenguaje principal que se usa para crear esas páginas web. Piénsalo como el esqueleto de una página: define qué elementos contiene y cómo están organizados.






¿Qué otros lenguajes se usan junto con HTML?


  • CSS (Cascading Style Sheets): para darle estilo (colores, tipografía, diseño).
  • JavaScript: para hacerlo interactivo (como botones, formularios dinámicos o animaciones).
